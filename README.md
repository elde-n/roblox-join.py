# roblox-join.py
A python script for roblox that launches the roblox-player

## Getting Started
```sh
python main.py -u user-name -p place-name
```

## Adding Users
Edit the '~/.config/accounts' file and add a newline containing "user:cookie"

## Adding Games
Edit the '~/.config/games' file and add a newline containing "place-id:alias"
